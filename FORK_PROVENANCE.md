# Fork Provenance

- Recorded on: 2026-06-24

## Dataset

The transcript data lives under `data/` and contains 297 `.txt` podcast transcript files, alongside RAG tooling for Qdrant, LightRAG, Streamlit, and graph exploration.

Product Career Ops should treat this repository as an external reference dataset. When using transcript-derived ideas in career artifacts, cite the source repository and transcript filename, summarize principles in your own words, and avoid copying raw transcript text into public application materials.
